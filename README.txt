
=== Stellar Luxe Website ===

Ce site contient :
- index.html : la page principale
- style.css : le design
- dossier images/ : contient background.png

=== Modifier les véhicules ===
Dans index.html, cherchez la section <table> sous "Nos véhicules".
Ajoutez ou supprimez les lignes comme ceci :
<tr>
    <td>Nom du véhicule</td>
    <td>Prix</td>
    <td>Description</td>
</tr>

=== Mettre en ligne ===
Téléversez tout sur GitHub Pages (n'oubliez pas le dossier images/).

=== Contact ===
Pour toute aide, modifiez le texte directement dans index.html.
